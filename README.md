# algo-practice

Solutions to algorithmic problems I've worked through. Mostly Python.

Each file is one problem with the description as a docstring at the top, my solution, and a short note on time/space complexity.

## Layout

- `arrays/` - array and string problems
- `trees/` - tree and graph problems
- `dp/` - dynamic programming
- `sliding_window/` - sliding window technique
- `graphs/` - BFS/DFS/shortest path
- `heaps/` - priority queue problems

## Why

Keeping the muscle memory alive. Re-doing classic problems every few months
is cheap and surprisingly useful for general engineering thinking.
